Seletores CSS

Seletor de filho direto: Seleciona um elemento que é filho direto, ou seja está diretamente após algum elemento específico na hierarquía do documento. O título (h1) acima é estilizado usando esse seletor.

Seletor de classe: Seleciona todos os elementos que compartilham o mesmo valor do atributo class. Esse parágrafo tem um background cinza usando esse seletor.

Seletor de id: Seleciona um elemento pelo seu identificador exclusivo. Esse parágrafo tem um background coral usando esse seletor.

Seletor de Atributo: Seleciona todos os elementos que tem um atributo setado para um valor específico. Muito usado em inputs. Esse parágrafo tem o bakcground em vermelho usando esse seletor.

Seletor de descendência: Seleciona todos os elementos que são filhos de outro elemento. Todos os parágrafos nessa página, que são filhos de body, tem seu tamanho de fonte alterado usando esse seletor.

Seletores de tipo: Seleciona todos os elementos que compartilham a mesma tag. Todos os parágrafos nesta página são estilizados (cor azul escuro e negrito) usando este seletor.
